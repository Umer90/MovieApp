# MovieApp
Assignment of movies listing and search.

## Prerequisites
A running system having macOS with Xcode.

## Installing
Download or clone project in-order to open it on Xcode and run project directly.

## External Libraries
1. AFNetworking
2. MBProgressHUD
3. SDWebImage
4. Reachability

## Helper Classes
1. SharedHelper
2. NetworkHelper
